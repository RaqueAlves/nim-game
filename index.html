Sobre o Projeto

Este projeto recria o jogo Nim em versão web interativa, aplicando conceitos de programação web e lógica matemática.

Objetivos

  • Aplicar conceitos de HTML5, CSS3 e JavaScript.
  • Implementar lógica do jogo manipulando o DOM.
  • Treinar organização de arquivos e uso de GitHub.
  • Explorar estratégias matemáticas do Nim.

Tecnologias e Ferramentas

  • HTML5
  • CSS3
  • JavaScript
  • Git & GitHub
  • MongoDB (planejado)

Recursos Usados

  • Editor VS Code
  • Testes em Chrome e Firefox
  • Documentação W3C e MDN
